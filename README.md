# Personal AI Effectiveness Assistant

Личный помощник для достижения мечты - веб-приложение для ежедневного планирования с иерархической системой целей и автоматической оценкой через Anthropic Claude API.

## Возможности

- **Иерархия целей** (7 уровней): День → Неделя → Месяц → Квартал → Полугодие → Год → Мечта (5 лет)
- **Ежедневное планирование**: создание плана утром, ввод факта вечером
- **ИИ-оценка дня** по 4 критериям:
  - Стратегическое развитие
  - Операционное управление
  - Работа с командой
  - Эффективность времени
- **Анализ выравнивания (alignment)**: проверка того, как задачи дня работают на цели всех уровней
- **История и аналитика**: отслеживание прогресса во времени

## Технологии

- **Frontend**: Next.js 15, React 19, TypeScript, Tailwind CSS
- **Backend**: Next.js API Routes
- **База данных**: SQLite с Prisma ORM
- **AI**: Anthropic Claude API (claude-sonnet-4.5)

## Установка и запуск

### 1. Установите зависимости

```bash
npm install
```

### 2. Настройте окружение

Создайте файл `.env.local` и добавьте ваш API ключ Anthropic:

```env
DATABASE_URL="file:./dev.db"
ANTHROPIC_API_KEY="sk-ant-ваш-ключ-здесь"
```

### 3. Создайте базу данных

**Важно**: В текущей среде Prisma binaries могут быть недоступны. Для создания базы данных вручную:

```bash
# Если sqlite3 установлен
sqlite3 prisma/dev.db < prisma/migrations/20251109_init/migration.sql
```

Или дождитесь запуска в окружении с доступом к Prisma CLI:

```bash
npm run prisma:migrate
npm run prisma:generate
```

### 4. Запустите приложение

```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:3000

## Использование

### 1. Установите цели
- Перейдите в раздел "Цели"
- Заполните мечту на 5 лет
- Установите цели для каждого периода (год, полугодие, квартал, месяц, неделя)

### 2. Планируйте день
- Перейдите в раздел "Сегодня"
- Утром создайте план на день
- Вечером добавьте факт выполнения

### 3. Получите оценку
- После заполнения плана и факта нажмите "Получить оценку от ИИ"
- Система отправит запрос к Claude API
- Вы получите оценку по 4 критериям и общую оценку
- ИИ проанализирует выравнивание ваших задач с целями всех уровней

## База данных

Проект использует SQLite с 5 таблицами:

- `dream_goal` - мечта на 5 лет
- `period_goals` - цели на периоды (неделя, месяц, квартал, полугодие, год)
- `daily_entries` - ежедневные записи (план и факт)
- `evaluations` - оценки от ИИ
- `open_tasks` - незакрытые задачи

## Скрипты

- `npm run dev` - запуск в режиме разработки
- `npm run build` - сборка для продакшена
- `npm run start` - запуск продакшен сборки
- `npm run prisma:generate` - генерация Prisma Client
- `npm run prisma:migrate` - применение миграций БД
- `npm run prisma:studio` - открыть Prisma Studio для просмотра БД

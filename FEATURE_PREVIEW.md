# 🎨 Превью: Динамические поля в блоке "Личные интересы"

## Что нового?

Теперь вы можете добавлять **собственные разделы** в блок "Личные интересы" с полноценными текстовыми полями для ввода.

---

## Как это выглядит

### До изменений ❌
```
┌─────────────────────────────────────────┐
│ Личные интересы                         │
├─────────────────────────────────────────┤
│ Хобби:   [__________________]           │
│ Спорт:   [__________________]           │
└─────────────────────────────────────────┘
```
**Только 2 фиксированных поля**

---

### После изменений ✅
```
┌─────────────────────────────────────────┐
│ Личные интересы                         │
├─────────────────────────────────────────┤
│ Хобби:   [чтение, программирование]     │
│ Спорт:   [бег, плавание]                │
│                                          │
│ Музыка:  [рок, джаз, классика] [Удалить]│
│ Книги:   [фантастика, детективы] [Удалить]│
│ Путешествия: [Европа, Азия]  [Удалить]  │
│                                          │
│ ┌────────────────────────────────────┐  │
│ │ [Название раздела] [+ Добавить раздел] │
│ └────────────────────────────────────┘  │
└─────────────────────────────────────────┘
```
**Неограниченное количество разделов!**

---

## Как использовать

### 1️⃣ Добавление нового раздела
1. Прокрутите до блока "Личные интересы"
2. Внизу найдите серый блок с полем ввода
3. Введите название раздела (например: "Музыка", "Кино", "Кулинария")
4. Нажмите **"+ Добавить раздел"** или **Enter**

### 2️⃣ Заполнение раздела
1. После создания раздела появится текстовое поле
2. Введите нужную информацию
3. Все данные сохраняются вместе с профилем

### 3️⃣ Удаление раздела
1. Нажмите кнопку **"Удалить"** справа от названия раздела
2. Раздел исчезнет (без подтверждения)

## Технические детали

### Хранение данных
Дополнительные разделы сохраняются в поле `customInterests` в формате JSON:
```json
[
  { "label": "Музыка", "value": "рок, джаз, классика" },
  { "label": "Книги", "value": "фантастика, детективы" },
  { "label": "Путешествия", "value": "Европа, Азия" }
]
```

### Изменённые файлы
- `prisma/schema.prisma` - добавлено поле `customInterests`
- `app/api/profile/route.ts` - обновлён API для работы с JSON
- `app/profile/page.tsx` - новый UI для управления разделами
